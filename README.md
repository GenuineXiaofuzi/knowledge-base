# Personal Knowledge Base

> Systematically organize learning, work, and creative content for cross-platform migration

## Folder Structure

| Folder | Purpose | Examples |
|--------|---------|----------|
| `Programming/` | Programming techniques, code optimization | Python async, Java concurrency |
| `GitHub/` | Open source analysis, Trending | Agent Skills, tool reviews |
| `AITools/` | AI tips, Prompt engineering | ChatGPT tricks, WorkBuddy workflows |
| `WeChat/` | Article writing, content creation | Viral article templates, writing tips |
| `Automation/` | Workflows, scheduled tasks | GitHub Trending scraper, auto-publish |
| `WorkBuddy/` | WorkBuddy usage tips | Conversation classification, skill development |
| `Daily/` | Daily pain points, problem-solving | WorkBuddy tips, automation workflows |

## File Naming Convention

```
YYYY-MM-DD-topic-description.md
```

Examples:
- `2026-05-09-GitHub-Agent-Skills-Analysis.md`
- `2026-05-09-WorkBuddy-Conversation-Classification.md`

## Sync Strategy

- **Local**: Markdown format for easy migration to any platform
- **GitHub**: Auto-sync with version control
- **IMA**: Key content synced to cloud knowledge base

## Usage

1. At the end of each WorkBuddy conversation, auto-prompt for classification
2. Save content as Markdown to corresponding folder
3. Auto Git commit + push to GitHub
4. Optional: Upload key content to IMA knowledge base
