# AI 角色蒸馏：让任何人成为你的智能助手

> 当 AI 能够学习并复现任何人的思维方式和表达风格，我们正在见证一场关于知识传承和人机协作的革命。今天为大家介绍两个开源项目——**女娲.skill** 和 **dot-skill（同事.skill）**。

---

## 🔥 项目一：女娲.skill - 蒸馏顶尖人物的思维方式

![女娲.skill Hero动画](https://github.com/alchaincyf/nuwa-skill/raw/main/assets/hero.gif)

### 项目简介

**女娲.skill**（nuwa-skill）的核心理念是：**「你想蒸馏的下一个员工，何必是同事。蒸馏任何人的思维方式——心智模型、决策启发式、表达DNA。」**

这个项目受 colleague-skill 启发，但将蒸馏场景从同事扩展到了全领域的顶尖公众人物。

### 核心能力

女娲真正**提取认知操作系统**，而非简单的角色扮演：

- **表达DNA** - 语气、节奏、用词偏好
- **心智模型** - 认知框架、思维方式
- **决策启发式** - 判断标准、决策逻辑
- **反模式** - 价值观底线、明确不做什么
- **诚实边界** - 明确标注能力局限

### 效果示例

**场景：精力管理问题**
```text
用户      ❯ 我同时想做自媒体、写书、做独立开发，精力完全不够

Naval     ❯ 你列了三个欲望。每个欲望都是你跟不快乐签的一份合同。
            不是精力不够，是合同太多。
            问你自己：哪一个做起来你会忘记时间？
            那个就是你的特定知识所在。
            不是三选一。是先一，再一，再一。
            串行复利，不是并行内耗。
```

> **这不是角色扮演**，而是用名人的认知框架帮你分析问题。

### 工作原理

输入一个名字后，女娲自动完成：

1. **六路并行采集** - 著作、访谈、社交媒体、批评者视角等6个Agent同时跑
2. **三重验证提炼** - 观点必须跨2+领域出现、能推断新问题的立场、有排他性
3. **构建Skill** - 心智模型 + 决策启发式 + 表达DNA + 价值观 + 诚实边界
4. **质量验证** - 用该人物公开回答过的问题测试，验证准确性

## 🧬 项目二：dot-skill（同事.skill）- 任意角色蒸馏引擎

### 项目简介

**dot-skill**（前身是 colleague-skill）是一个通用的角色蒸馏引擎：**任何人都可以被蒸馏成 AI Skill**——同事、伴侣、家人、朋友、偶像、公众人物、虚构角色，甚至你自己。

### 三大角色家族

| 家族 | 适用场景 | 核心能力 |
|------|----------|----------|
| 🧑‍💼 **colleague** | 同事、导师、队友 | 工作Skill+人格两层架构 |
| 💞 **relationship** | 伴侣、家人、朋友 | 表达DNA、情绪触发点、冲突模式 |
| 🌟 **celebrity** | 公众人物、创作者 | 六维度研究工具链，还原思维模型 |

### 效果示例

**🌟 公众人物——蒸馏 Andrej Karpathy**
```text
User            ❯ Everyone's talking about Agents now — what do you think?

celebrity.skill ❯ Everyone's doing demos, nobody's doing eval.
                  The Agent bottleneck was never model intelligence — it's scaffolding:
                  a strict evaluation harness, fine-grained error replay.
                  Most products are still stuck at "wrap an API + a few tools."
                  That's a demo, not an agent.
```

**🧑‍💼 同事——蒸馏字节跳动后端工程师**

```text
User            ❯ Can you review this API design?

colleague.skill ❯ Hold on — what's the impact? You haven't explained the context.
                  (after reading) N+1 query, fix it. Use the standard
                  {code, message, data} response format. That's the spec.
```

### 多数据源支持

| 数据源 | 消息 | 文档 | 说明 |
|--------|------|------|------|
| 🟢 **飞书**（自动） | ✅ | ✅ | 输入名称即可完全自动采集 |
| 🟡 **钉钉**（自动） | ⚠️ | ✅ | 钉钉API不支持消息历史采集 |
| 🟣 **Slack**（自动） | ✅ | — | 需要管理员安装Bot |
| 💬 **微信聊天** | ✅ | — | 需先用工具导出 |
| 📄 **PDF/图片** | — | ✅ | 手动上传 |

### 多Agent宿主兼容

dot-skill 生成的Skill可以一键安装到4类主流Agent宿主：

- 🟣 **Claude Code** - 原生支持斜杠命令
- 🟠 **Hermes Agent** - 一键安装
- 🔵 **OpenClaw** - 完全兼容
- ⚫ **Codex** - 通过Skill名称调用

### Skill进化能力

- 📥 **追加文件** → 自动分析差异 → 合并到对应章节
- 💬 **对话修正** → 输入"他们不会这么做" → 写入修正层
- 🕰️ **版本控制** → 每次更新自动归档，可回滚
- 🔬 **质量检查** → 全流程质量校验

### 安装使用

打开你的 Claude Code/Hermes/OpenClaw/Codex，输入：
```
Install the dot-skill skill for me: https://github.com/titanwings/colleague-skill
```

完成后输入 `/dot-skill` 即可启动。

## 🤔 两个项目的关系与对比

| 对比维度 | 女娲.skill | dot-skill |
|---------|------------|-----------|
| **核心定位** | 蒸馏公众人物的思维方式 | 通用角色蒸馏引擎（任何人） |
| **擅长场景** | 公众人物、创作者、思想家 | 同事、亲密关系、公众人物 |
| **蒸馏深度** | 五层认知框架（思维操作系统） | 两层架构（工作Skill+人格） |
| **数据源** | 公开信息（著作、访谈、社交媒体） | 多源数据（飞书、钉钉、Slack、微信等） |
| **预置Skill** | 13位人物+1个主题 | 社区100+ Skill |
| **宿主兼容** | Claude Code | Claude Code/Hermes/OpenClaw/Codex |

**女娲.skill** 更适合想要学习顶尖人物思维方式、获取高质量决策建议的用户。

**dot-skill** 更适合想要保留特定人物（同事、朋友、家人）的交互风格、创建个性化AI助手的用户。
